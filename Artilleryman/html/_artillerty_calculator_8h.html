Detailed Description

Заголовочный файл модуля, в котором производятся рассчеты.

Angular Sweep Time Complexity: O(n*n*lon(n))

В этом файле перечислены функции и структуры, используемые для рассчета. Модуль реализует алгоритм угловой развертки.
